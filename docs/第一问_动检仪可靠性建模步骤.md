# 第一问：动检仪可靠性量化建模步骤（已实现版）

## 1. 目标
对每台设备（`设备ID=车型_车号`）计算可靠性分数，筛出异常设备。

- 输入：`csv_cleaned/*.csv`
- 输出：
  - `reliability_score`（0-100，越低越可疑）
  - `is_abnormal_device`（0/1）
  - 规则命中明细（用于解释）

实现脚本：`scripts/q1_cluster_model.py`

## 2. 当前数据规模
- 总记录数：369,328
- 设备数：308
- 时间范围：2024-01-01 到 2024-04-09

## 3. 建模单元与邻域定义
- 分析单元：设备级（每台设备一行特征）
- 同点群体：`线号 + 行别 + 50米网格里程 + 日期`
- 群体基准：同点群体的中位数（用于构造偏差）

## 4. 特征工程（代码中已落地）

### 4.1 系统偏差与一致性
- `peer_signed_residual_mean`：有符号偏差均值（区分偏大/偏小）
- `peer_residual_mean`：绝对偏差均值
- `peer_residual_p95`：绝对偏差P95
- `peer_level_disagree_rate`：与群体等级不一致比例

### 4.2 非共识高报警
- `l3plus_rate`：三级及以上占比
- `isolated_l3plus_rate`：三级及以上中“孤立报警”比例
- `l3plus_cnt`：三级及以上次数

### 4.3 时间漂移
- `weekly_mean_slope`：周均值全局斜率（取绝对值）
- `piecewise_mean_gap`：后半段均值与前半段均值差（取绝对值）
- `rolling_max_slope`：4周窗口局部最大斜率

### 4.4 横纵向失衡
- `vh_ratio_mean`：垂向/横向加速度比值均值
- `vh_ratio_std`：比值波动
- `vh_level_gap_rate`：垂/横等级差>=2的比例

### 4.5 波动与质量
- `max_acc_std`：主加速度波动
- `max_acc_p95`：主加速度P95
- `conflict_rate`：等级冲突占比
- `n`、`coverage_days`：样本量与覆盖天数
- `dominant_vehicle_type`：主车型（用于分层标准化）

## 5. 聚类与分数计算

### 5.1 分层标准化
按 `dominant_vehicle_type` 分组做 `RobustScaler`；组样本过少时回退全局标准化。  
目的：降低不同车型动力学差异带来的伪异常。

### 5.2 DBSCAN 参数选择
- `min_samples = max(5, floor(sqrt(设备数)))`
- 用 `NearestNeighbors` 计算 k-distance
- 在多个分位点（0.75~0.95）生成 `eps` 候选
- 评分准则：有有效簇、噪声率接近目标、主簇规模更大
- 选最优 `eps` 后执行 `DBSCAN`

### 5.3 可靠性分
- 找最大簇作为“主正常簇”
- 计算每台设备到主簇中心的欧氏距离
- 距离线性映射为 `reliability_score`（越远分越低）

## 6. 异常判定（双门槛）
设备满足任一条件即判异常：

1. `is_noise == 1`（DBSCAN噪声点）
2. `reliability_score <= P10`
3. 规则命中数 `rule_hit_count >= 2`

规则项（各自用P95作阈值）：
- `peer_residual_p95`
- `isolated_l3plus_rate`
- `weekly_mean_slope`
- `rolling_max_slope`
- `vh_level_gap_rate`
- `max_acc_std`

## 7. 输出文件
输出目录：`outputs/q1/`

- `q1_device_features.csv`：设备特征表
- `q1_device_scores_cluster.csv`：分数、聚类标签、规则命中
- `q1_abnormal_devices_cluster.csv`：异常设备清单
- `q1_cluster_summary.txt`：本次运行摘要（参数、数量、Top设备）

## 8. 运行方式
```bash
python scripts/q1_cluster_model.py
```

## 9. 当前已知局限
- 小样本设备仍可能被噪声化，后续可增加“证据不足”单独标签。
- 目前采用单次聚类结果，建议增加时间切分稳定性评估（例如前后两月对比）。
