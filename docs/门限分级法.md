速度门限分类策略（通俗版）

我们的目标是：让同样的振动数据，在不同车速、不同车型下都能被公平判断，不会“一把尺子量到底”。

1. 先分组，再判断  
把数据先按“车型”和“车速区间”分开。  
原因：不同车型、不同速度下，振动本来就不一样，必须分开看才合理。

2. 每组都做4个门限  
我们把状态分成5个等级（0到4级），所以需要4条分界线（t1,t2,t3,t4）。  
以后拿到一个新数据，就看它落在哪个区间里，直接给等级。

3. 门限怎么定（尽量稳）  
优先用“稳健分位数”定门限：  
- 低一级的高位值（95%分位）  
- 高一级的低位值（5%分位）  
这两者取中点作为边界。  
这样不容易被少量异常点带偏。

4. 数据不够时的兜底  
如果某个小组样本太少：  
- 先用“同车型整体门限”补  
- 还不够再用“全局门限”补  
保证每个组都有可用门限，不会算不出来。

5. 最后做一次合理性修正  
强制门限满足：t1 <= t2 <= t3 <= t4。  
避免出现“后面的门限反而更小”的错误情况。

6. 实际分类规则  
给定一个指标值 x：  
- x < t1：0级  
- t1 <= x < t2：1级  
- t2 <= x < t3：2级  
- t3 <= x < t4：3级  
- x >= t4：4级

一句话总结  
这是一个“按车型+车速分层、样本不足自动回退、门限稳健可解释”的分级策略，特点是简单、实用、便于工程落地。
