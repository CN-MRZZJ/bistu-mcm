# 第三问 Writeup：按公里区间的线路质量评价

## 1. 问题目标
第三问要求基于检测数据，以 1 公里区间为单位（例如 100.000-100.999），评价线路质量，并给出质量最好的 10 个区间和最差的 10 个区间。

核心目标是把原始逐条检测记录，稳定映射为区间级质量分数，避免被偶发尖峰和小样本波动误导。

## 2. 数据与分组定义

### 2.1 分析对象
- 输入数据：`csv_cleaned` 中全部清洗后记录。
- 主字段：`线号`、`行别`、`里程`、`max_acc`、`数据等级`、`车速`、`日期`。

### 2.2 公里区间定义
按如下规则构造区间编号：

`公里区间 = floor(里程)`

并以 `线号 + 行别 + 公里区间` 作为唯一评价单元。

## 3. 指标构建
对每个公里区间聚合以下统计量：

1. `样本量`：区间记录条数。  
2. `max_acc均值`：整体振动平均水平。  
3. `max_accP95`：高位风险强度（主指标）。  
4. `max_accP99`：极端风险参考。  
5. `三级及以上占比`：高等级报警频度。  
6. `车速均值`、`覆盖天数`：辅助解释指标。  

其中 `is_l3_plus = 1(数据等级>=3)`，用于计算高等级占比。

## 4. 评分模型

### 4.1 分组稳健标准化
由于不同线路/行别的基线差异明显，先在每个 `线号+行别` 组内做稳健标准化：

`z(x) = (x - median(x)) / MAD(x)`

对 `max_acc均值`、`max_accP95`、`三级及以上占比` 分别计算 z 值。

### 4.2 原始风险分
采用加权线性组合：

`原始风险分 = 0.20*z(max_acc均值) + 0.55*z(max_accP95) + 0.25*z(三级及以上占比)`

权重设计考虑：
- `P95` 体现区间高风险主体，权重最高；
- 均值反映整体状态；
- 高等级占比提供告警侧验证。

### 4.3 小样本收缩
为降低小样本区间偶然偏高/偏低造成的误判，引入样本权重：

`样本权重 = clip(样本量 / 组内样本量中位数, 0, 1)`

`风险分 = 原始风险分 * 样本权重`

### 4.4 统一风险指数
将 `风险分` 转为全体可比较的百分位指数：

`质量风险指数 = rank_pct(风险分) * 100`

定义区间质量等级：
- A(优)：0-20  
- B(良)：20-40  
- C(中)：40-60  
- D(差)：60-80  
- E(很差)：80-100  

指数越低越好，越高越差。

## 7. 方法评价与改进方向
当前方案优势：
- 分组稳健标准化，减少跨线基线偏差；
- 兼顾高位风险与告警频度；
- 小样本收缩降低波动误判。

可进一步改进：
1. 引入邻近公里平滑（空间连续性约束）。  
2. 融合车型/车速结构差异做二次校正。  
3. 若有现场检修标签，可做监督学习优化权重。
