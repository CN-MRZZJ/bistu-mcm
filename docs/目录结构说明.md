# 项目目录结构说明

## 当前结构
```text
附件数据/
├─ csv_raw/                # 原始CSV数据
├─ csv_cleaned/            # 清洗后CSV数据
├─ xlsx/                   # 原始xlsx数据
├─ reports/                # 清洗报告等文本报告
├─ outputs/
│  └─ q1/                  # 第一问建模输出结果
├─ scripts/                # 所有脚本
│  ├─ data_cleaning.py
│  ├─ file_converter.py
│  └─ q1_cluster_model.py
└─ docs/                   # 题干与建模文档
   ├─ 题干.txt
   ├─ 第一问_动检仪可靠性建模步骤.md
   └─ 目录结构说明.md
```

## 运行建议
- 清洗：`python scripts/data_cleaning.py`
- 第一问聚类：`python scripts/q1_cluster_model.py`

## 结果位置
- 清洗后数据：`csv_cleaned/`
- 清洗报告：`reports/`
- 第一问结果：`outputs/q1/`
